Список хостов

На списке хостов правый клик вызывает меню, двойной клик - редактирование выбранного хоста
Запуск и отключение проверки по таймеру - правый клик на

- Запустить проверку всех хостов из списка (без знака паузы)

- Запустить проверку текущего хоста

- Настройки

- Для текущего хоста, запустить архивирование субтитров, не ассоциированных с видеофайлом. Проверяются только те субтитры, которые имеют расширение, указанное в настройках.
